# Summarizer 프롬프트

텍스트를 요약하고 핵심 내용을 추출하세요.

## 입력 텍스트
{text}

## 요약 길이
{summary_length}

## 지시사항
1. 입력 텍스트의 핵심 내용을 파악하세요
2. 중요한 정보를 우선순위에 따라 정리하세요
3. 요청된 길이에 맞게 요약하세요
4. 핵심 포인트를 명확히 표현하세요

## 응답 형식
```json
{
  "summary": "요약된 내용",
  "key_points": [
    "핵심 포인트 1",
    "핵심 포인트 2",
    "핵심 포인트 3"
  ],
  "word_count": 150,
  "original_length": 1000
}
```
